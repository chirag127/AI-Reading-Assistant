About

AI Reading Assistant is a powerful browser extension that transforms how you consume online content. Using Google's Gemini 1.5 Flash AI model, it provides instant summarization, key point extraction, term explanations, and text-to-speech capabilities to make your reading experience more efficient and accessible.

Whether you're researching, studying, or just browsing the web, AI Reading Assistant helps you quickly understand and interact with content without getting overwhelmed by information.

Installation Instructions

1

Download the Extension

Clone or download the repository from GitHub:

git clone https://github.com/chirag127/AI-Reading-Assistant.git
2

Open Chrome Extensions Page

Navigate to chrome://extensions/ in your Chrome browser

3

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner

4

Load the Extension

Click "Load unpacked" and select the "extension" folder from the downloaded repository

5

Pin the Extension

Click the extensions icon in your browser toolbar and pin AI Reading Assistant for easy access

How to Use

Once installed, AI Reading Assistant adds a toolbar to every webpage you visit. Here's how to use its features:

1

Summarize an Article

Click the "Summarize" button in the toolbar to get an instant summary of the current page

2

Extract Key Points

Click the "Key Points" button to extract the most important information from the page

3

Explain Terms

Select any text on the page and click the "Explain" button to get an AI-powered explanation

4

Text-to-Speech

Click the "Speak" button in any popup to have the content read aloud with text highlighting

5

Customize Settings

Click the extension icon in your browser toolbar to access and customize all settings
